Merge Sort Visualizer

Merge Sort is a divide-and-conquer algorithm that recursively divides the input array into two halves, sorts each half, and then merges the two sorted halves to produce the final sorted array. It efficiently handles large datasets and has a time complexity of O(n log n). The merging step is key, where two sorted arrays are combined into a single sorted array by comparing the smallest elements of each.This visualizer demonstrates the Merge Sort algorithm.
