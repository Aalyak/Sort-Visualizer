Insertion Sort Visualizer

Insertion Sort is a simple comparison-based sorting algorithm that builds the final sorted array one element at a time. It iterates through the input array and, for each element, inserts it into its correct position relative to the elements before it. This process continues until the entire array is sorted. Insertion Sort is efficient for small datasets and nearly sorted arrays, with a time complexity of O(nÂ²) in the average and worst cases.
