Selection Sort Visualizer

Selection Sort is a straightforward comparison-based sorting algorithm that works by repeatedly finding the smallest (or largest) element from the unsorted part of the array and swapping it with the first unsorted element. This process continues, moving the boundary between the sorted and unsorted portions of the array until the entire array is sorted. Selection Sort has a time complexity of O(nÂ²) and is generally less efficient than other algorithms like Quick Sort or Merge Sort for large datasets, but it is easy to understand and implement.
